**Phase II Armor** was the second-generation clone trooper body armor
introduced midway through the Clone Wars. It improved on Phase I design
with better ergonomics, enhanced sensor packages, and a modular
attachment system. The design strongly foreshadowed the stormtrooper
armor of the Galactic Empire.

# Specifications

| Property         | Value                           |
|------------------|---------------------------------|
| Material         | Plastoid alloy composite        |
| Protection       | Blaster bolts, shrapnel, vacuum |
| Helmet Features  | HUD, comlink, air filtration    |
| Modular Slots    | Pauldron, kama, backpack        |
| Weight           | ~12 kg                          |
